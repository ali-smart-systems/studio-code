# نظام تتبع اليد والوجه (Computer Vision System)

هذا المشروع عبارة عن أداة برمجية مكتوبة بلغة Python تقوم بتتبع حركات اليد وملامح الوجه بشكل دقيق ومباشر.

## المميزات:
- تتبع حركة اليد (Hand Tracking) بدقة عالية.
- اكتشاف ملامح الوجه (Face Detection).
- معالجة مباشرة للبيانات (Real-time processing).

## التقنيات المستخدمة:
- Python
- OpenCV
- MediaPipe
